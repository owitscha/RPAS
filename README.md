# Reddit Post Archiving Script
## Description
This Script is used to Archive Reddit posts. Essentially it opens all the Comments and starts the printing process via the browser.
*Info: This Script was created with the help of Claude AI*
## How to use
- Open the Reddit-Post URL
- Open the browser Dev-Tools (F12-Key)
- Go to the *Console*-Tab
- Copy the content from the RPAS.js file into the console and run the sript (Enter-Key)
- Wait until the print-dialogue pops up --> Here select the *Save as PDF*-Option as Destination
- Click print and select a path
